Conclusion

The NFS server implementation of the NFS V2 protocol with UDP protocol in the 2.2 series Linux kernel plus the kernel patches in knfsd 1.4.2 works with Solaris and Linux NFS clients. With the multi-threaded kernel NFS server improves the NFS server performance dramatically. It shows that Linux on the adequate hardwares makes a viable NFS server option for enterprise.
