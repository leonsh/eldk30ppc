Other Factors

The NFS server related programs depend on hostname lookup for correct operation. glibc 2.1.1 or above is recommended for better hostname lookup performance. If NIS/NIS+ is used, the Name Switch Cache Daemon (nscd) in glibc should be started to further improve hostname lookup speed.
