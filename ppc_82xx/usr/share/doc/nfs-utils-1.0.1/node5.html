Interoperability

To deploy Linux as an NFS server in heterogeneous multivendor environment, we have to ensure that the Linux NFS server can interoperate with different NFS client implementations. We used the kernel-space NFS server in Linux kernel 2.2.9 and the user-space NFS server of version 2.2beta37 to test interoperability of the Linux NFS server implementations.
